# One God's Power - Site (pronto para GitHub Pages)

Ficheiros incluídos:
- `index.html` - página principal
- `style.css` - estilos
- `script.js` - funcionalidades (alternar idioma, scroll suave)
- `imagens/` - imagens de exemplo (SVGs)
- `README.md` - este ficheiro

## Como usar
1. Cria uma conta no GitHub (https://github.com) se ainda não tiveres.
2. Cria um novo repositório público com o nome que desejares (ex: `onegodspower`).
3. Faz upload de todos os ficheiros desta pasta para o repositório.
4. Vai a **Settings → Pages** e ativa GitHub Pages na branch `main` (ou `master`).
5. O site ficará disponível em `https://<teu-username>.github.io/<nome-do-repo>` em alguns minutos.

## Como editar (rápido)
- Para mudar textos: abre `index.html` e procura o texto dentro das secções.
- Para mudar autor: edita `script.js` (linha authorNameSpan.textContent) ou o rodapé no `index.html`.
- Para adicionar imagens: substitui os ficheiros em `imagens/`.
- Para adicionar uma nova secção: copia uma `<section>` e adapta.

## Nota sobre o nome do autor
No ficheiro `index.html` o autor aparece como **Nagi Kira** (nome de pena). Se quiseres manter privado o teu nome real, é seguro usar um pseudónimo em sites públicos.
